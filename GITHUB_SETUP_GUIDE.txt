================================================================================
TROUBLESHOOTING GITHUB
================================================================================

PROBLEM: "Permission denied (publickey)"
SOLUTION: Setup SSH key
$ ssh-keygen -t ed25519 -C "your-email@example.com"
$ eval "$(ssh-agent -s)"
$ ssh-add ~/.ssh/id_ed25519
Then add public key to GitHub Settings → SSH and GPG keys


PROBLEM: "Merge conflict"
SOLUTION: 
1. Pull latest: $ git pull
2. Fix conflicts in files
3. $ git add .
4. $ git commit -m "Resolve merge conflict"
5. $ git push


PROBLEM: "File too large"
SOLUTION:
GitHub has 100MB file limit. If needed:
1. Use Git LFS (Large File Storage)
2. Or split large files
3. Or remove from repo (for generated files)


PROBLEM: "Accidentally committed sensitive info"
SOLUTION:
$ git rm --cached .env
$ echo ".env" >> .gitignore
$ git commit -m "Remove .env from tracking"
$ git push


PROBLEM: "Need to undo last commit"
SOLUTION:
$ git reset --soft HEAD~1
(Keeps changes, uncommits)


PROBLEM: "How to sync forked repo"
SOLUTION:
$ git remote add upstream https://github.com/ORIGINAL-OWNER/repo.git
$ git fetch upstream
$ git merge upstream/main
